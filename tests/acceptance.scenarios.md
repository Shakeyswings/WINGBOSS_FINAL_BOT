# Acceptance scenarios

## Customer path (DM)
1. /start
2. Tap Start Order → Browse Menu
3. Add item to cart (scaffold)
4. Checkout → Share Location → send location
5. Upload payment proof photo
6. See Status

## Staff path (group)
1. Owner runs /staff_test_order (in DM)
2. Staff group receives order card with buttons
3. Tap Accept → Cooking → Ready → Book Driver → Picked → Delivered
4. Tap Dispatch Pack and verify links/address hints

## Academy (DM)
1. /academy
2. Set role Fryer
3. Quick drill → answer
4. Attempt is appended to data/staff_quiz_attempts.json
